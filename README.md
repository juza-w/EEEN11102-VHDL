# EEEN11102 Digital Electronics VHDL Labs
This repository contains a template for the Year 1 Digital Electronics VHDL Labs.
It installs and configures the Open Source EDA tools, including TerosHDL (Visual Studio Code Extension), GHDL, Surfer and Yosys.

## PLEASE READ ME 
When you first start machine, it will take some time build the codespace. After a while, you should see something like this on bottom of the screen :

```bash
✓ Finishing up...
⠋ Running postCreateCommand...
  › bash .devcontainer/setup-tools.sh
```

Please wait until that process finishes, Be patient, it may take a couple minutes. It is only needed the first time round, once the machine is configured the start up should be much quicker.

### Important
- TerosHDL (the software you will be using throughout your exercises) is preconfigured for the lab exercises, running on GitHub Codespaces. This is the recommended way of working for these labs - just open this template in a Codespace (in a browser window, or on a local copy of Visual Studio Code). Remeber to Deactivate a Codespace when you finish working. You can return to the Codespace to continue your work at any time.
  
- The  software can be installed on your own computer (Linux, or even Windows), but if you want to do this you will need to follow additional instructions to install and configure the software.

- The lab files you create will be stored in your Codespace. It is recommended that you follow the usual practice of committing and synchronising with the repository (your local copy of the EEEN11102-VHDL repository), so  that the files are available to you on GitHub, even outside of the Codespace.
